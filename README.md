# Weather Dashboard Pro 🌦️

## Overview

Weather Dashboard Pro is a desktop application developed using Python, Tkinter, and OpenWeatherMap API. It provides real-time weather information for any city around the world.

---

## Features

- Search Any City
- Real-Time Weather Data
- Temperature Information
- Humidity Information
- Wind Speed Information
- Weather Conditions
- Professional GUI

---

## Technologies Used

- Python 3
- Tkinter
- Requests
- REST API

---

## Installation

```bash
pip install requests
pip install pillow
```

---

## Run

```bash
python main.py
```
